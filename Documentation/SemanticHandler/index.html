Class Semantic​Handler 

public class SemanticHandler : CustomStringConvertible

The SemanticHandler is an special classs that manages the semantic actions of the compiler.

This class is initilizated in the parser and uses the informacion of certail tokens or expresions to execute some functions

insert​Symbol​ToST(_:​_:​_:​_:​_:​params:​) 

public func insertSymbolToST(_ id : NSString, _ constant: Bool, _ array : Bool, _ type: TypeSymbol = .void, _ kind : Kind = .field, params: [Symbol] = [])

This function inserts in the symbol table an identifier and its properties, like the type, kind, and more. If it wasn't possible to insert it to the symbol table, it sends an error indicating that the variable was already declared.

When it was inserted it asks for a local or global address in memory depending on the deepth of the symbol table

Parameters

Name Type Description
id NSString

The identifier to be inserted in the symbolTable
constant Bool

Add a flag indicating if the symbol is constant
array Bool

Add a flag indicating if the symbol is an array
type Type​Symbol

Add the type of the symbol, by default is void
kind Kind

Add the kind of the symbol (field or method), by default is a field
params [Symbol]

The symbols that are received as parameters in case of a function, the default value is an empty array []
